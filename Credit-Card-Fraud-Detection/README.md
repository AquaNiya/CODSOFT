# Credit Card Fraud Detection

This project focuses on identifying fraudulent credit card transactions using machine learning.

## What I worked on
- Loaded and inspected the transaction dataset
- Understood class imbalance
- Normalized the data
- Trained classification models
- Evaluated the model using precision, recall, and F1-score

## What I learned
This task showed me why fraud detection is different from regular classification problems, especially because accuracy alone is not enough.

## Dataset
Source: Kaggle Credit Card Fraud Detection Dataset
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
